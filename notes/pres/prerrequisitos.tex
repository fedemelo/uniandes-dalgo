\part*{Prerrequisitos}

Estos apuntes están escritos de forma que los prerrequisitos para entenderlos son sorprendemente laxos para el nivel de material que se cubre. Hay que saber lo siguiente:
\begin{itemize}
  \item Programación básica en algún lenguaje de programación, incluyendo recursión. 
  \begin{itemize}
    \item El seudo código que se emplea es bastante parecido a Python.
  \end{itemize}
  \item Algo de teoría sobre las estructuras de datos más básicas: arreglos, listas, etc. En qué difieren y cuáles son las ventajas de una sobre otra.
  \item Notación matemática básica (conjuntos, operadores, etc.), aritmética básica (sumar, multiplicar, etc.), álgebra elemental (despejar variables, factorizar, etc.) y funciones elementales (logaritmos, exponenciales, etc.).
\end{itemize}
